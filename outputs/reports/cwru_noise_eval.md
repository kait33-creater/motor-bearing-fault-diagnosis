# CWRU 抗噪鲁棒性评估报告

## 设计

模型用干净信号训练,测试信号叠加不同信噪比(SNR)的高斯白噪声,模拟现场传感器噪声。SNR 越低噪声越强。

## 结果

| SNR | 准确率 |
|---|---|
| 无噪声 | 1.0000 |
| 10dB | 1.0000 |
| 6dB | 1.0000 |
| 2dB | 1.0000 |
| 0dB | 0.9409 |
| -2dB | 0.4816 |
| -4dB | 0.3045 |

## 结论

无噪声时准确率 1.000;在最强噪声(-4dB,噪声功率大于信号)下仍达 0.304。准确率随噪声平滑衰减,说明所提特征对噪声有一定鲁棒性,其中频域与无量纲时域特征贡献了主要的抗噪能力。
