# 数据字段说明

本项目使用电机运行状态数据，主要字段如下：

| 字段名 | 含义 | 单位/说明 |
|---|---|---|
| timestamp | 采样时间 | 时间戳 |
| motor_id | 电机编号 | M001、M002 等 |
| voltage | 电压 | V |
| current | 电流 | A |
| temperature | 温度 | ℃ |
| vibration | 振动强度 | mm/s 或模拟指标 |
| speed | 转速 | rpm |
| load_rate | 负载率 | % |
| status | 运行状态 | 正常 / 预警 / 故障 |
| fault_type | 故障类型 | 正常 / 过载 / 过热 / 轴承异常 / 电压波动 |

## 状态设计思路

- 正常状态：各项运行参数处于合理范围；
- 预警状态：部分参数接近异常阈值；
- 故障状态：一个或多个关键参数明显异常。

## 故障类型设计思路

- 过载：电流和负载率持续偏高；
- 过热：温度明显升高，可能伴随电流偏高；
- 轴承异常：振动强度升高，转速波动增大；
- 电压波动：电压不稳定，并可能引起电流波动。
