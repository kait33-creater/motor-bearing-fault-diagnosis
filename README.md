# 基于 Python 的电机运行状态监测与故障预警系统

## 1. 项目简介

本项目面向工业电机运行状态监测场景，基于 Python 构建电机运行数据分析与故障预警系统。项目通过电压、电流、温度、振动、转速、负载率等运行参数，对电机状态进行分析，实现正常、预警、故障状态识别，并输出故障类型与维护建议。

## 2. 项目目标

- 构建电机运行状态数据集；
- 完成数据读取、清洗与可视化分析；
- 设计基于规则阈值的故障预警方法；
- 使用机器学习模型进行故障类型识别；
- 搭建 Streamlit 可视化看板；
- 输出项目报告和演示材料。
